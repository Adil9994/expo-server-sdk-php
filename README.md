## Installation

You can install the package via composer:

```bash
composer require ctwillie/expo-server-sdk-php
```

## Usage

```php
$expo-server-sdk-php = new Twillie\Expo();
echo $expo-server-sdk-php->echoPhrase('Hello, Spatie!');
```
